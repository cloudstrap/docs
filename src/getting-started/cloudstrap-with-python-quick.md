#### 1. Създай приложение

    app create mypythonapp python-3.3

#### 2. Промени кода

Промени с предпочитан редактор файла wsgi.py между 'body' таговете:

    <body>
        <h1>Hello world!</h1>
    </body>

#### 3. Качи приложението

    git push

#### 4. Виж в браузър

    http://mypythonapp-<mydomain>.sapp.io/


**Да, толкова е лесно!**
