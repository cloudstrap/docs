#### Валидни маркери за NodeJS приложение
Име на маркера Какво прави?
use_npm Инициализира твоето приложение чрез 'npm start' използвайки информацията от supervisor servername.js (където 'servername.js' е стойността, която си задал в package.json’s `main секцията).
hot_deploy Забранява рестартирането на приложението по време на git push.
