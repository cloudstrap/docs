#### Валидни маркери за всяко едно PHP приложение/уеб сайт
Име на маркера Какво прави?
force_clean_build Ще изтрие всички dependencies и ще започне да ги инсталира на ново, при всеки git push
hot_deploy Apache сървърът няма да се рестартира по време на build/deploy
disable_auto_scaling Ще забраните автоматичното скалиране на приложението
