#### Валидни маркери за PHP приложения
Име на маркера Какво прави?
force_clean_build Ще изтрие всички dependencies и ще ги инсталира, при всеки git push
hot_deploy Уеб сървърът няма да се рестартира при на качване на приложението
disable_auto_scaling Забранява автоматичното скалиране на приложението
use_composer Стартира автоматично инсталирането на Composer при всеки build. За подробности Composer.
enable_public_server_status Разрешава 'server status page' да бъде публично достъпна
