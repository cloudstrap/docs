#### Валидни маркери за Ruby приложение
Име на маркера Какво прави?
force_clean_build Ще задейства чист re-bundle по време на build цикъл.
hot_deploy Забранява рестартирането на приложението по време на git push. The Passenger 'Restart.txt' файл ще бъде използван, за да презареди заявката.
disable_asset_compilation Забранява прекомпилирането на assets. Този маркер трябва да се използва при публикуване на приложения с вече компилирани assets.
