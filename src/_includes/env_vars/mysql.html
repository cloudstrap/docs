#### MySQL ENV променливи
Име на променливата Какво съдържа?
OPENSHIFT_MYSQL_DIR Пътя в който MySQL съхранява данните си.
OPENSHIFT_MYSQL_DB_HOST HOST/IP на MySQL сървъра. Важно: Имайте в предвид, че ако приложенито не е скалируемо, IP адресът е локален за хост сървъра, което означава, че няма да може да бъде достъпен извън него.
OPENSHIFT_MYSQL_DB_PORT Порта на който работи MySQL сървъра
OPENSHIFT_MYSQL_DB_USERNAME Потребилското име с което можете да достъпвате базата си
OPENSHIFT_MYSQL_DB_PASSWORD Паролата с която можете да достъпвате базата си
OPENSHIFT_MYSQL_DB_SOCKET Пътя до сокет файла на сървъра
OPENSHIFT_MYSQL_DB_URL URL до вашия MySQL сървър от вида: mysql://[user]:[pass]@[host]:[port]/
OPENSHIFT_MYSQL_DB_LOG_DIR Път до директорията в която се записват логовете на MySQL
