{% section id="action-hooks" title="Action Hooks" %} Това са скриптове със специфични имена, които се изпълняват от системата според твоите нужди. Те могат да са написани на Python, PHP, Ruby, shell и други. StartApp поддържа следните `action hooks`.
Action Hook предназначение
pre_build Изпълнява се локално на твоята CI система, ако е налична. В противен случай, се изпълнява преди build стъпката
build Изпълнява се локално на твоята CI система, ако е налична. В противен случай, се изпълнява преди deploy стъпката
deploy Изпълнява се след като dependencies са обработени, но преди да е стартирано приложението
post_deploy Изпълнява се след deploy и стартиране на приложението
#### Добавяне на Action Hook Пример: echo "echo 'Hello from deploy Action Hook'" > .openshift/hooks/deploy action hook script файла трябва да се направи executable: chmod +x .openshift/hooks/deploy Запиши промените в `Git`: git commit -am "Add deploy action hook" Качване на сървъра: git push {% endsection %}
